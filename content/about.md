# 关于我

这里是一块为长期写作预留的小地方。

我会在这里记录三类内容：

1. 技术实践：搭建、重构、工具链与前端细节
2. 文字草稿：尚未成熟，但值得保留的想法
3. 日常观察：那些不会立刻消失的小经验

## 为什么要做这个站点

我希望博客的维护方式足够简单：

- 写文章时，不需要打开复杂后台
- 部署时，不需要维护服务器
- 回看时，所有内容都有稳定的版本记录

所以我选择了 `Vue + Element Plus + Markdown + GitHub Pages` 这组组合。

它不是最重的方案，但足够稳定，也足够清晰。

## 关于设计

界面会尽量保持克制：

- 少量颜色
- 清晰层级
- 合理留白
- 优先阅读

如果你也在搭建属于自己的博客，希望这里能给你一点参考。
